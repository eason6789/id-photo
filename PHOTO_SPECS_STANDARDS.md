# 证件照规格与标准规范

## 📐 通用规范要求

### 1. 头部占比规范
- **头部高度**: 应占照片总高度的 **60-70%**
- **头顶到照片顶部**: 2-5mm 空间
- **下巴到照片底部**: 7mm 以上空间
- **示例**: 对于 35mm 高的照片，头部（头顶到下巴）应为 21-24.5mm

### 2. 头部位置规范
- **水平居中**: 头部左右居中，偏移不超过 ±2mm
- **垂直位置**: 眼睛应在照片高度的 **55-60%** 位置（从底部算起）
- **头部倾斜**: 不超过 ±5°

### 3. 头肩比例
- **肩宽**: 应占照片宽度的 **65-75%**
- **颈部可见**: 至少显示 5-10mm 的颈部
- **肩部对称**: 两肩水平，高度差不超过 2mm

### 4. 面部要求
- **正面直视**: 面部完全正对相机，不能侧脸
- **双眼睁开**: 眼睛自然睁开，不能闭眼或斜视
- **嘴巴闭合**: 自然闭嘴，露出微笑可以，但不能露齿大笑
- **面部表情**: 自然平静，不能夸张

### 5. 头发要求
- **露出额头**: 刘海不能遮挡眼睛和眉毛
- **露出耳朵**: 至少一只耳朵可见（部分国家要求）
- **头发不凌乱**: 整齐，不能遮挡面部轮廓

### 6. 眼镜要求
- **允许佩戴**: 可以戴眼镜，但必须满足以下条件：
  - 镜框不能遮挡眼睛
  - 无反光（不能有明显光斑）
  - 能清晰看到双眼
  - 非墨镜、非彩色镜片
- **建议**: 如果眼镜反光严重，建议摘下

### 7. 着装要求
- **深色服装**: 建议穿深色衣服（与背景形成对比）
- **无高领**: 不能穿高领衣服遮挡下巴
- **无帽子**: 不能戴帽子、头巾（宗教原因除外）
- **无饰品**: 不能戴夸张的耳环、项链等

### 8. 光线与清晰度
- **均匀光线**: 面部无阴影，光线均匀
- **清晰度**: 面部特征清晰可见，无模糊
- **无红眼**: 不能有红眼效应
- **肤色自然**: 不能过度美白或失真

## 🎯 AI 生成需要实现的规范检查

### 当前缺失的功能

1. **头部占比检查**
   - 检测人脸位置和大小
   - 计算头部占照片的比例
   - 如果不符合 60-70%，调整缩放

2. **头部位置调整**
   - 检测眼睛位置
   - 确保眼睛在照片高度的 55-60% 位置
   - 自动上下平移调整

3. **水平居中**
   - 检测人脸中心
   - 确保左右居中

4. **头部倾斜校正**
   - 检测双眼位置
   - 计算倾斜角度
   - 自动旋转校正（不超过 ±5°）

5. **肩部检测**
   - 确保肩部可见
   - 肩宽占 65-75%

6. **质量检查**
   - 模糊度检测
   - 光线均匀度检测
   - 背景纯色检测

## 💡 实现建议

### 方案1: 使用 OpenCV 实现规范检查

```python
import cv2
import numpy as np

def check_id_photo_compliance(image_path):
    """检查证件照是否符合规范"""
    img = cv2.imread(image_path)
    height, width = img.shape[:2]
    
    # 1. 人脸检测
    face_cascade = cv2.CascadeClassifier(haarcascade_frontalface_default.xml)
    faces = face_cascade.detectMultiScale(img, 1.1, 4)
    
    if len(faces) == 0:
        return {"compliant": False, "error": "未检测到人脸"}
    
    if len(faces) > 1:
        return {"compliant": False, "error": "检测到多个人脸"}
    
    x, y, w, h = faces[0]
    
    # 2. 头部占比检查
    head_ratio = h / height
    if head_ratio < 0.60 or head_ratio > 0.70:
        return {
            "compliant": False,
            "error": f"头部占比 {head_ratio:.1%}，应在 60-70% 之间"
        }
    
    # 3. 水平居中检查
    face_center_x = x + w // 2
    image_center_x = width // 2
    offset = abs(face_center_x - image_center_x)
    if offset > width * 0.05:  # 偏移超过 5%
        return {
            "compliant": False,
            "error": f"人脸未居中，偏移 {offset}px"
        }
    
    # 4. 眼睛位置检查
    eye_cascade = cv2.CascadeClassifier(haarcascade_eye.xml)
    eyes = eye_cascade.detectMultiScale(img[y:y+h, x:x+w])
    
    if len(eyes) < 2:
        return {"compliant": False, "error": "未检测到双眼"}
    
    # 计算眼睛平均高度
    eye_y = np.mean([eye[1] for eye in eyes]) + y
    eye_position_ratio = (height - eye_y) / height
    
    if eye_position_ratio < 0.55 or eye_position_ratio > 0.60:
        return {
            "compliant": False,
            "error": f"眼睛位置 {eye_position_ratio:.1%}，应在 55-60% 之间"
        }
    
    return {"compliant": True, "message": "符合规范"}
```
